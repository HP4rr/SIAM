# SIAM
System Integrity and Anomaly Monitor, a real time security focused system monitor in python

## Current status (v0.2 - Dec. 2025)
- Current CPU usage percentage
- Current memory usage percentage
- Total storage, free storage, and used storage
- Live process list with PID, name, and status
- 0.2s refresh
- Ctrl+C exit

## Future roadmap
1. CPU/RAM top 5 processes (instead of the process list)
2. Network connections
3. Curses/TUI dashboard
4. File system monitoring
5. Rule-based + ML anomaly detection

Target finish date: (Spring 2027)
